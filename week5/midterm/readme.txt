Project Title: Quiz Game

How to Run the Program

    Open the project in the terminal by typing ./main

    Follow the on-screen instructions to take the quiz.

Program Description

This is a console-based multiple-choice quiz game that asks the user 3 questions, each with 4 possible answers. The user selects an answer by typing the letter (A–D). After all questions are answered, the program displays the final score.

Extra Notes

    Input is case-insensitive for answers (i.e., "a" and "A" are both valid).

    The questions and answers can be expanded by modifying the questions list.
